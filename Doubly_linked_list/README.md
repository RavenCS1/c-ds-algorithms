# Doubly Linked List

## Build from **this directory**

**CMake:**  
cmake -S . -B build  
cmake --build build  
cd build  
./Doubly_linked_list  

**Makefile (recommended - clean build):**  
make build-clean  
./Doubly_linked_list  

**`make build-clean` builds binary + removes .o files (only binary remains)**
